# Örnek HTML Form

## ```test.php``` Dosyasının İçeriği

```PHP
<?php
  echo "<pre>";
  echo '<h1>$_POST Değişkenin Değeri</h1>';
  print_r($_POST);
  echo '<h1>$_GET Değişkenin Değeri</h1>';
  print_r($_GET);
?>
```
